# Tugas Pemrograman lanjut Teknik Komputer 2020

Header terletak pada folder include  
Isinya ada pada folder lib  
Main code ada di folder src  
Kepentingan hiasan dan lain lain ada pada folder asset  
Folder game berisi objek hasil compile

Compile & run:

```
./run
```

run:

```
./play
```

`*nb untuk update compile tinggal edit pada file bernama 'run'`
